          CNF Requirements
        

        

          
A → BC (two non-terminals)

          
A → a (single terminal)

          
S → ε (only if ε is in L(G))

        

      

      

        

          
G

          GNF Requirements
        

        

          
A → a(alpha) (terminal first)

          
alpha is a string of non-terminals

          
S → ε (only if ε is in L(G))

        

      

    


    
    

      

        

          
        

        Input Format Guide
      

      

        

          
Non-terminals

          
A, B, C, S

          
(uppercase letters)

        

        

          
Terminals

          
a, b, c

          
(lowercase letters)

        

        

          
Production arrow

          
-> or →

          
(separator)

        

        

          
Epsilon

          
e or ε

          
(empty string)

        

      

      

        
Example Grammar:

        
S -> AB | a
A -> aA | e
